同域跨窗口事件触发器演示

请在不同标签页打开以下页面：

  • on - 监听
  • on - 监听，3秒轮询一次
  • once - 只监听一次

输入消息之后，点击按钮发送，就可以看到其它标签页的结果。




代码

<input type="text" placeholder="请输入" id="msgInput" />
<button id="sendBtn">点击发送消息</button>

<script src="../dist/cross-window-emitter.min.js"></script>
<script>
  var msgInputEl = document.getElementById("msgInput");
  var sendBtnEl = document.getElementById("sendBtn");

  sendBtnEl.onclick = function () {
    crossWindowEmitter.emit('msg', msgInputEl.value);
  }
</script>
